## 📖 Sobre a Disciplina
Este repositório tem como objetivo principal o embasamento teórico e a compreensão dos conceitos fundamentais da ciência da computação. O foco inclui a análise matemática e lógica de algoritmos, bem como a compreensão de como as estruturas de dados são organizadas e manipuladas na memória.

---

## 🗂️ Conteúdos e Fundamentos

Cada tópico listado abaixo aborda os conceitos, propriedades e análises assintóticas correspondentes:

| Tópico / Estrutura | Descrição Teórica |
| :--- | :--- |
| **Ponteiros e Memória** | Endereçamento, ponteiros e alocação dinâmica (Pilha vs. Heap). |
| **Listas Lineares** | Organização, operações e diferenças estruturais (sequenciais e encadeadas). |
| **Pilhas (Stacks)** | Propriedade LIFO e aplicações (ex: inversão e expressões pós-fixas). |
| **Filas (Queues)** | Propriedade FIFO, filas estáticas, dinâmicas e circulares. |
| **Recursividade** | Princípio de indução, recorrência e pilhas de execução (Torres de Hanói). | 
| **Análise de Complexidade** | Notação Big-O, análise de pior/melhor caso e custo computacional. |
| **Ordenação** | Fundamentos matemáticos e complexidade de algoritmos (Bubble, Insertion, Merge, Quick). |
| **Busca** | Comparação teórica e limites de busca linear e binária. |
